# SunCart – Summer Essentials Store

SunCart is a simple summer online shop website.  
Users can see summer products like sunglasses, skincare items, beach accessories, and summer clothes.

## Project Purpose

This project is made for learning **Next.js**, **Tailwind CSS**, and **BetterAuth**.

Users can:

- see products
- see product details
- login and register
- view profile

---

## Main Features

- Summer theme homepage
- Responsive design for mobile, tablet, and desktop
- Product data from JSON file
- Protected product details page
- Login and register system
- Google login
- User profile page
- Update profile information

---

## Technologies Used

- Next.js
- React
- Tailwind CSS
- DaisyUI
- BetterAuth
- Animate.css
